General information

    License management system

    In this release the dongles are optional since a new license management system is used. When installing the product with a QuickStart key it will be usable for 30 days. During this period the product(s) must be registered with IAR to activate the license. After registration a permanent key will be sent to you which opens up the product(s) for which you have a valid license. The product will be locked to the computer on which it has been installed unless a License Server or dongle is used. A stand-alone license can be transfered to a different PC by using the license transfer functionality. For more information, see Installation and license activation above.
